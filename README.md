# Node.js Random Color Generator

Generate a colored block with # characters on the command line

## Random colored block

```bash
$ node index.js
  ###############################
  ###############################
  ###############################
  ######                   ######
  ######      #abedf4      ######
  ######                   ######
  ###############################
  ###############################
  ###############################
```

## Colored block by hue and Luminosity

```bash
$ node index.js dark green
  ###############################
  ###############################
  ###############################
  ######                   ######
  ######      #0fa844      ######
  ######                   ######
  ###############################
  ###############################
  ###############################
```
